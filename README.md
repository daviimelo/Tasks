# Tasks App

Um sistema completo de gestão de tarefas (To-Do), desenvolvido com arquitetura monorepo dividida entre um aplicativo mobile e uma API RESTful. O foco principal é oferecer uma interface limpa e responsiva para organização pessoal.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

### Frontend (Mobile)
*   **React Native** (Framework para construção da interface nativa)
*   **TypeScript / JavaScript** (Linguagem base)

### Backend (API)
*   **Node.js** (Ambiente de execução)
*   **Express** (Micro-framework para roteamento da API)

## Estrutura do Projeto

O repositório está organizado em duas partes principais:

*   `/ProjectTasks` - Código-fonte do aplicativo React Native.
*   `/TasksBackend` - Código-fonte da API em Node.js/Express.
